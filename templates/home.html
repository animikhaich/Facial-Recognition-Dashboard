Facial Recognition Demo

This is my full stack weekend project. It performs Facial Recognition using Keras Implementation of VGG Face by RCMalli. The source code for this can be found in the GitHub repo HERE. 

Please follow the following instructions as we proceed forward:

  1. Upload pictures of the people whose faces you would like to recognize in the future
  2. Faces will be detected and you will be prompted to enter the names of each detected face. You can skip a face if you don't want it to be identified.
  3. Upload the query images (different form the first ones) where the same group labeled group of faces are present.
